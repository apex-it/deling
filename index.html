deling.js

deling.js is a simple social share plugin with minimal fuzz. deling.js is also easy to customize with options and CSS.

Usage

Include deling.js and deling.css in your HTML. Add the class deling to any element that should get the sharing icons/texts.

usage.js will by default use the current URL as the URL to share. You can override/provide your own by adding the data-url attribute.

Usage after Pageload/DOMContentloaded

If you have dynamic pages or elements inserted late you can use deling(domElement); to make deling.js create sharing for your elements.

Example: Defaults

Insert all networks with defaults. 

<div class="deling"></div>

Example: Names

Insert all networks with names. 

<div class="deling" data-names="true"></div>

Example: No icons

Insert all networks without icons. 

<div class="deling" data-icons="true"></div>

Example: Custom URL

Insert all networks with a custom URL. 

<div class="deling" data-url="https://www.heimdalit.no/"></div>

Example: Some networks

Insert some networks with a custom URL. 

<div class="deling" data-networks="twitter,google-plus" data-url="https://www.heimdalit.no/"></div>

Example: Custom icon classes

Almost plug and play with font-awesome as icon font instead of bundles. 

<div class="deling" data-url="https://www.heimdalit.no" data-icon-class="fa fa-"></div>

Example: custom title, media and summary

For networks that support or require either of these you can customize with data-title, data-media and data-summary. 

<div class="deling"
    data-title="deling.js a delicious share plugin"
    data-media="http://somewhere/that/is/a/image.png"
    data-summary="This is a customized summary.">
  </div>

Example: disable colored icons

You can disable the colored icons by using data-color="false". Making deling.js not add "brand" color to the icons. 

<div class="deling" data-color="false"></div>

Example: wrap anchor in different element

You can have deling.js wrap it's anchors in a different element if you wish. This makes it ideal to use with any framework you use. 

<ul class="deling" data-anchor-wrap="li"></ul>

    Example: add extra CSS class to button element

    If you need to specify extra CSS class(es) to the button for any reason you can with data-button-class. 

    <div class="deling" data-button-class="awesome-o"></div>
